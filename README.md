Finance Diary (Updated Version)

A simple and user-friendly Finance Diary application to help track daily income, expenses, and savings. This updated version provides a cleaner interface and improved financial record management.

Features
Add income and expense entries
Track daily financial activities
View total income, expenses, and balance
Simple and organized dashboard
Easy-to-use interface
Data management for better budgeting
Tech Stack
HTML
CSS
JavaScript
Purpose

This project was built to help users monitor their personal finances, understand spending habits, and make better financial decisions.
